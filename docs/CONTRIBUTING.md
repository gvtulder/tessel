# Contributing to Tessel

Hello! Tessel is made by me, Gijs van Tulder, and available under a GPL-3.0 license.

If you find any bugs or have other suggestions, you are very welcome on [GitHub](https://github.com/gvtulder/tessel/issues).

## Translations

Translations of Tessel are managed through [Weblate](https://hosted.weblate.org/projects/tessel/). You can easily create an account and add translations for existing or new languages.
