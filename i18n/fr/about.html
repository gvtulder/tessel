Comment jouer ?

  • Faites glisser une tuile de la pile pour la placer sur le plateau. Cliquez sur une tuile pour la tourner.
  • Faites correspondre les couleurs avec les tuiles à côté.
  • Complétez une forme fermée d'une seule couleur pour gagner des points. Plus la forme est grande, plus vous recevez de points.

Le jeu se termine quand vous avez placé toutes vos tuiles.

Envie d’un défi ?

Vous avez joué avec les triangles et les carrés ? Essayez d’autres formes et motifs de tuiles dans une partie personnalisée.

Vous voulez mélanger les choses ? Changez les règles ; exigez que les couleurs ne correspondent pas, ou utilisez seulement des tuiles aux couleurs uniques.

Remarque : certaines combinaisons peuvent rendre impossible de faire un motif de tuiles valide ! Le jeu vous avertira si cela arrive.
